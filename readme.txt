Basic Instruction
=================

1	Create a database with a user with the details as stored in the database.php file (or your own
	user connection details)
	
2	Import the SQL file (gmap.sql) provided to create the tables and populate with initial data

3	Upload the application and system folders outside of your public web root

4	Upload the www folder to your public web root

5	Go to your web address and enter <your url>/index.php/admin. When prompted enter the username
	admin and password internet to login.
	
6	You should now be logged into the system
